Dark Rectangles Theme for NodeBB
=========================

A simple dark theme for NodeBB.

Based on [Lavender Theme for NodeBB](https://www.github.com/psychobunny/nodebb-theme-lavender).

### Installation

    npm install nodebb-theme-dark-rectangles

### Credits
